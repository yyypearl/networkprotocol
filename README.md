# Network Protocol : '네트워크로 말해요' 게임

# 구현 목적
- 오프라인 게임을 온라인 게임으로 구현
- 게임의 각 단계의 매끄러운 연결 및 진행
- 프로토콜을 통한 전반적인 통신 시스템 이해
=> ‘네트워크로 말해요’ 게임 만들기

# 기대효과 및 활용 방안
-	비대면 프로그램의 활성화로, 시공간의 제약을 받지 않고 다양한 게임으로 응용 가능ex. 교육용, 레크레이션 등
-	세대를 아우를 수 있는 융합형 게임 제작
